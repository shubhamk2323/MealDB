Meal DB App

Overview:

    The Meal DB App is an Android application that fetches and displays meal details using Retrofit to interact with an external API.
    The app efficiently handles API responses and presents dynamic meal data to users.

Features:

    Fetch meal data from an external API using Retrofit.

    Display meal details dynamically in the app.

    Handle API responses and parse JSON data efficiently.

    Implement MVVM architecture with LiveData and ViewModel.

Technologies Used:

    Kotlin - Primary language for Android development.

    Retrofit - For API requests and response handling.

    MVVM Architecture - Ensures clean architecture and maintainability.

    ViewModel - For efficient UI updates and data handling.

Installation & Setup

    Clone this repository

    Open the project in Android Studio.

    Sync Gradle dependencies.

    Run the application on an emulator or physical device.

Usage

    Open the app to fetch meal data from the API.

    View detailed information about each meal.

    Ensure an active internet connection for fetching real-time data.

Future Enhancements

    Implement search functionality for meals.

    Add favorite meals feature with Room Database.
